# MTProto Proxy + Fake TLS


Простой и удобный установщик **MTProto прокси** с **Fake TLS** за Cloudflare.  
Скрипт создан специально для новичков — всё устанавливается в диалоговом режиме (с вопросами).
В репозитории оставлены только необходимые файлы:

### Особенности
- Красивая анимация загрузки при заходе на домен в браузере (выглядит как защищённое соединение)
- Fake TLS (маскировка под обычный HTTPS-трафик)
- Автоматическая настройка Nginx + Docker + Firewall
- Одна команда установки после подготовки
- Работает на Ubuntu 22.04 / 24.04
- Минимальная задержка (пинг) с проводного и мобильного интернета
- `install.sh` — основной рабочий установщик.
- `README.md` — инструкция.

### Как это работает
- При обычном заходе в браузер на ваш домен — показывается стильная анимация «Connecting to secure network...»
- Telegram подключается по специальной ссылке `tg://proxy?...` и использует **Fake TLS** для маскировки трафика
- Домен прячется за Cloudflare (оранжевое облако)
## Запуск

**Важно**: Fake TLS не меняет реальный провайдера IP на Cloudflare (это технически невозможно для MTProto). Но правильный выбор fake-домена улучшает маскировку и иногда снижает пинг.

---
```bash
bash <(curl -fsSL https://raw.githubusercontent.com/GamesFanTeam/MTProto-Proxy/main/install.sh)
```
